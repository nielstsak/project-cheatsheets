Module de référence technique.

1. Requêtes DQL (Data Query Language)

SELECT id, name, email 
FROM users 
WHERE created_at > '2023-01-01' 
ORDER BY name DESC;

Extraction filtrée et triée.

2. Jointures (Joins)

SELECT orders.id, users.name
FROM orders
INNER JOIN users ON orders.user_id = users.id;

Intersection stricte entre deux tables relationnelles.
