GIT

Module de référence technique.

1. Flux Standard

git init
git add .
git commit -m 'feat: initial commit'
git push origin main

Cycle de vie basique d'un commit et synchronisation distante.

2. Branching & Merging

git checkout -b feature/login   # Créer et changer
git merge develop               # Fusionner
git rebase main                 # Réécrire l'historique

Stratégies de divergence et de réconciliation.
